# AI-learning-resources

This repository contains a **static, no-login website** that curates a focused reading path through important AI papers.

The site is designed for GitHub Pages and helps learners move from core foundations to modern frontier work in a structured way.

## What The Website Includes

- A phase-by-phase roadmap of 18 papers
- Direct links to original paper sources (mostly arXiv)
- Clean, minimal UI inspired by modern developer-doc style pages
- Responsive layout for both laptop and mobile screens
- Fully static content with **no forms, no login, and no data collection**

## Who It Is For

- Students starting AI/LLM reading seriously
- Developers who want a practical paper sequence
- Anyone looking for a quick, structured AI paper roadmap
